Este site está a ser criado para, além de explorar outra linguagem de programação, servir para melhorar a pesquisa binária.

E o que é uma pesquisa binária?

A pesquisa binária é um algoritmo eficiente de "divisão e conquista" utilizado para localizar um item numa lista ordenada. Em vez de verificar cada elemento um a um (busca linear), ele reduz o espaço de pesquisa pela metade a cada etapa, tornando a busca extremamente rápida.

Como Funciona

  1. O meio: O algoritmo localiza o elemento que está exatamente no centro da lista.
  2. A comparação: Compara o elemento procurado (alvo) com o elemento central.
  • Se o alvo for igual ao meio, o item é encontrado e a busca termina.
  • Se o alvo for menor que o meio, a busca continua apenas na metade esquerda (valores menores).
  • Se o alvo for maior que o meio, a busca foca apenas na metade direita (valores maiores).
